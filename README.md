# RC Performance Dashboard v2

## Deploy to Vercel

1. Push to GitHub
2. Import repo in Vercel
3. Deploy (no environment variables needed)

## Local dev

```bash
npm install
npm start
```

## Features
- Password protected (car2026)
- Apps Script URL saved to localStorage (no re-entry needed)
- Auto-refresh every 30s
- 4 tabs: Performance, Ads, Product Health, Inventory
- Timerange selector: MTD / Last 30D / Last Month / YTD
